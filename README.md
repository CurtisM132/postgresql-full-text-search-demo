<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

* Docker Installation
* Go Installation

### Installation

1. Clone repository
2. Run _docker-compose up_ in the *db* folder
3. Build executable using _go build_
4. Run executable
5. Open browser to localhost:4200
